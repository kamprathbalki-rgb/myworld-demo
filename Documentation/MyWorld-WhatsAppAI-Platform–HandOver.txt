MyWorld WhatsApp AI Platform – Handover Document

Project Objective

The system monitors selected WhatsApp groups, classifies messages using AI, extracts structured real-estate information, and stores the results in MongoDB for matching, recommendations, and SaaS administration.

High Level Flow

WhatsApp Group Message
        ↓
whatsapp-web.js
        ↓
createClient.js
        ↓
classifyWhatsappMessage()
        ↓
extractWithAI()
        ↓
AI JSON Output
        ↓
Store in MongoDB
        ↓
Inventory / Requirement Records
        ↓
Future Matching Engine

Important AI Output Format

Current AI contract:

{
  "classification": "INVENTORY",
  "multipleRecords": true,
  "records": [
    {
      "transactionType": "RENT",
      "propertyType": "APARTMENT",
      "bhk": 2,
      "location": "Kharadi",
      "budget": 30000
    }
  ],
  "contactNumbers": [],
  "confidence": 0.99,
  "summary": ""
}

DO NOT revert to:

{
  "propertyType": "",
  "bhk": ""
}

All future development must use: records[] architecture.

Rule 2

Rule 3

first.
