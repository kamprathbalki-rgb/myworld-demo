Company Name

What he can view - Currently only: - Company registrations

Purpose: Run the CRM for their company.

Step-1 Register Admin User - /register

Enter :-
Name
Email
Password
Role = admin
Tenant ID (given by SaaS admin)

Step-2 Login - /login

Step-3 Main links after login
/dashboard/main
/dashboard/admin-dashboard
/dashboard/analytics
/dashboard/intelligence

Manage Properties :-
/property/add

/property/list
/property/map-view

Title
Property Type
Price

/buyer/add
/buyer/page

Input:-
Name

Purpose: Handle leads and property visits.

Register - /register

Input:-
Name
Email
Password
Role = executive
Tenant ID

Login - /login
